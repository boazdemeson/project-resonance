# Product Principles

Every feature must follow these principles.

## 1. Songs First

Everything revolves around songs.

Not folders.

Not files.

## 2. Reduce Chaos

Every feature should simplify collaboration.

## 3. Invisible Complexity

Powerful doesn't mean complicated.

Advanced functionality should feel natural.

## 4. Collaboration by Default

Every project is designed for multiple contributors.

## 5. Beautiful by Default

Musicians spend hours inside creative software.

Our interface should feel inspiring, not corporate.

## 6. Never Lose Work

Every revision should be recoverable.

Nothing should disappear accidentally.
