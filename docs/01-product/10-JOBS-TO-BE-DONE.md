# Jobs To Be Done

## Purpose

This document defines why musicians use the platform.

Every feature should satisfy one or more Jobs To Be Done.

---

## JTBD 1 — Capture Ideas

When inspiration strikes,

I want to create a Session immediately,

So I never lose an idea.

---

## JTBD 2 — Save Progress

When I finish working,

I want to save my current state,

So I can continue later without losing work.

---

## JTBD 3 — Remember Context

When I return after days or weeks,

I want to immediately understand where I left off,

So I can continue creating without confusion.

---

## JTBD 4 — Track Evolution

When my Session changes,

I want every Snapshot preserved,

So I can revisit any creative decision.

---

## JTBD 5 — Organize Naturally

When I create many Sessions,

I want to organize them without thinking about folders,

So my creative space stays clean.

---

## JTBD 6 — Gather Feedback

When I need another opinion,

I want collaborators to comment directly inside the Session,

So discussions stay connected to the music.

---

## JTBD 7 — Keep Everything Together

When I work on music,

I want lyrics, notes, files, and audio together,

So I never search across multiple applications.

---

## JTBD 8 — Resume Instantly

When I reopen a Session,

I want to immediately know:

- Latest Snapshot
- Recent activity
- Outstanding comments

So I can continue creating immediately.

---

## Decisions

Every feature must satisfy at least one Job To Be Done.
