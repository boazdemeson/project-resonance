# MVP Scope

## Purpose

Define the smallest product that delivers real value.

The MVP must solve one problem exceptionally well.

---

# Core Problem

Independent musicians lose context while creating music.

The MVP helps musicians preserve and organize creative work.

---

## Workspace

- Authentication
- User Profile

- Archive

## Success Criteria

Users say:

"I finally have one place for every creative session."

---

## Decisions

The MVP prioritizes solo musicians.

Collaboration can be introduced after validating the core experience.
