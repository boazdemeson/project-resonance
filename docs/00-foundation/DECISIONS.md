# Company Decisions

This document summarizes the current product direction.

For historical context and rationale, see the ADR folder.

---

# Mission

Build the best creative workspace for independent musicians.

---

# Core Entity

Session

Everything revolves around a Session.

---

# Product Philosophy

Create first.

Organize naturally.

Collaborate when needed.

---

# Primary User

Independent musicians.

Usually working alone.

Occasionally collaborating.

---

# What We Are

- Creative workspace
- Music creation companion
- Session management platform
- Creative memory system

---

# What We Are Not

- DAW
- Cloud storage
- Music distributor
- Streaming platform
- Generic project management tool

---

# Core Workflow

Create Session

↓

Capture Ideas

↓

Save Snapshots

↓

Continue Creating

↓

Collaborate (Optional)

↓

Complete Session

---

# Source of Truth

Every important piece of information belongs to the Session.

Music Projects never own creative history.

---

# Collaboration Philosophy

Solo-first.

Collaboration is an enhancement.

Not a requirement.

---

# Product Focus

Creation.

Not distribution.

Not publishing.

Not marketing.

---

# Success Metric

A musician can return to any Session months later and instantly understand:

- What happened
- Where they left off
- What to do next

---

# Long-Term Vision

Become the first application musicians open when they begin creating.

---
